50. Explain difference.

Question

👉 ConfigMap vs Secret

Storage
Encoding
Use cases


🔹 ConfigMap vs Secret
1. Storage
   ConfigMap → stores non-sensitive data in plain text
   Secret → stores sensitive data (credentials, tokens, keys)
2. Encoding
   ConfigMap → plain text (no encoding)
   Secret → base64 encoded (not encrypted by default)
3. Use Cases
 ConfigMap
   Application configuration
   Environment variables
   Config files
 Secret
   Passwords
   API keys
   TLS certificates
